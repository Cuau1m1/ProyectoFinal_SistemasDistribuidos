# ProyectoFinal_SistemasDistribuidos
PROYECTO SISTEMAS DISTRIBUIDOS FINAL

CASO DE ESTUDIO: SIMULACIÓN DE RED BITTORRENT

Este proyecto debe integrarse con  código fuente y ejecutables (en github) en conjunto con su reporte digital enviando un único enlace de drive para tal efecto, lo cual está indicado como una tarea en el Google Classroom de la unidad de aprendizaje.
El proyecto debe presentar las soluciones que se pueden implementar vía software para resolver problemas clásicos que ocurren en el escenario distribuido: Transparencia, concurrencia, tolerancia a fallos, etc
